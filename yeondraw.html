📌 Gemini API 키가 왜 필요해요?

  • 손글씨 수식을 AI가 자동으로 LaTeX 수식으로 바꿔줘요.
  • 구글 계정만 있으면 무료로 발급받을 수 있어요.
  • 한 번만 입력하면 브라우저에 저장되어 다음에는 바로 사용돼요.
  • 저장된 키는 이 컴퓨터에만 보관되며 외부로 전송되지 않아요.

📝 발급 순서

  1. 아래 버튼을 눌러 aistudio.google.com 접속 (VPN 필요할 수 있음)
  2. 구글 계정 로그인 후 "Create API key" 클릭
  3. 생성된 키(AIza...)를 복사
  4. 아래 칸에 붙여넣기 → "저장"

🟠 Claude API (Anthropic)

  • 손글씨 수식을 Claude가 LaTeX로 변환해 줘요.
  • Gemini가 차단된 환경에서 대안으로 사용 가능.
  • claude-haiku-3-5 모델 사용 — 빠르고 저렴해요.
  • 키는 브라우저에만 저장, 외부 전송 없어요.

📝 발급 순서

  1. 아래 버튼으로 console.anthropic.com 접속
  2. 계정 생성 / 로그인 후 "API Keys" 메뉴 클릭
  3. "Create Key" → 키(sk-ant-...) 복사
  4. 아래 칸에 붙여넣기 → "저장"

🟢 ChatGPT API (OpenAI)

  • 손글씨 수식을 GPT-4o-mini가 LaTeX로 변환해 줘요.
  • 빠르고 저렴한 모델을 사용해요.
  • 키는 브라우저에만 저장, 외부 전송 없어요.

📝 발급 순서

  1. 아래 버튼으로 platform.openai.com 접속
  2. 로그인 후 "API keys" → "Create new secret key"
  3. 생성된 키(sk-...) 복사
  4. 아래 칸에 붙여넣기 → "저장"
